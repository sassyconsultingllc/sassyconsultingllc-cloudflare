Security architecture

4-Layer Security Sandbox. Zero Google.

Security isn't a marketing checkbox — it's baked into the architecture. Pure Rust eliminates entire classes of vulnerabilities that haunt Chrome and Brave.

🦀 Memory Safety by Default

Written entirely in Rust — no null pointers, no buffer overflows, no use-after-free. The classes of bugs that power 70%+ of browser CVEs simply cannot exist in safe Rust code.

🔲 4-Layer Process Sandbox

Web content, file viewers, extensions, and AI each run in isolated sandboxes. A compromised webpage cannot reach your files. A malicious script cannot escape its layer.

🚫 Zero Google Dependencies

No Chromium. No V8. No SafeBrowsing. No telemetry. No sync servers. No Google APIs. Your browsing data exists only on your machine.

🛡️ Anti-Fingerprint Engine

Canvas, WebGL, AudioContext, and font enumeration fingerprinting are randomized or blocked. Advertisers cannot build a persistent profile of your device across sites.

🔐 Local Encryption

Passwords use ChaCha20-Poly1305 with Argon2id key derivation. Your vault key never leaves your machine. No cloud sync means no breach risk from our servers — because we don't have any.

📊 Network Transparency

Real-time network activity monitor shows every connection your browser makes. Nothing is hidden. See trackers blocked, requests made, and data transferred — all in plain view.

🕵️ Data Poisoning

Actively frustrates trackers that slip through the blocker by injecting false behavioral signals. Makes harvested data unreliable — the tracking still happens, but the profile is garbage.

📜 Sandboxed Scripts

User scripts run in an isolated context inside the browser process — not as external processes or loaded extensions. No extension attack surface. Scripts cannot reach system APIs.

Why I Built This

I spent 11 years as a Chemical Warfare Officer in the U.S. Army and National Guard, then earned a Chemistry degree from UW-Eau Claire. I know what it's like to need specialized scientific software that costs more than most people's monthly rent.

I watched researchers share cracked copies of ChemDraw because $2,600/year is prohibitive on a grad student stipend. I watched my parents get harassed by ads and browser hijackers because Chrome's extension ecosystem is a minefield. I watched small businesses pay $100/month for Microsoft 365 because there was no real alternative that just worked.

So I built one. In Rust. From scratch. With no compromises on privacy, security, or capability.

Sassy Browser is the browser I always wanted to exist. I think you'll want it too.

— Shane Smith, Founder & CEO, Sassy Consulting LLC

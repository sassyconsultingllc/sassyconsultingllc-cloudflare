Features

Professional forensics tools. No bloat.

Everything you need for Windows incident response in a single binary.

REG

Registry Analysis

Parse and analyze Windows registry hives. Extract user activity, installed programs, network connections, and persistence mechanisms.

REC

File Recovery

Recover deleted files from NTFS volumes. Carve files from unallocated space using signature-based detection.

TL

Timeline Generation

Build comprehensive timelines from multiple sources. Correlate events across registry, file system, and logs.

EV

Evidence Preservation

Create forensically sound copies with hash verification. Chain of custody documentation built in.

MFT

MFT Parser

Parse NTFS Master File Table for complete file system analysis. Extract timestamps, paths, and metadata.

LOG

Event Log Analysis

Parse Windows Event Logs (EVTX). Filter, search, and export security-relevant events.

Usage

Simple CLI. Powerful results.

# Analyze registry for persistence mechanisms
winforensics registry --hive C:\Windows\System32\config\SOFTWARE --persistence

# Generate timeline from multiple sources
winforensics timeline --mft C:\$MFT --evtx C:\Windows\System32\winevt\Logs\ --output timeline.csv

# Recover deleted files
winforensics recover --volume C: --output recovered/ --signatures jpg,png,pdf,docx

# Parse MFT for file system analysis
winforensics mft --input C:\$MFT --output mft_analysis.json

# Create forensic image with verification
winforensics image --source \\.\PhysicalDrive0 --output evidence.dd --verify

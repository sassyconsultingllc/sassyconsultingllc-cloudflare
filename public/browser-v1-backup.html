Security architecture

4-Layer Sandbox. Trust Is Earned, Not Given.

Every page starts sandboxed. No clipboard access, no downloads, no popups, no camera — until YOU decide to trust it.

🔒 Page Sandbox

Every page isolated in memory. 3 meaningful interactions before permissions unlock. Anti-clickjacking, anti-bot, ad container detection — all automatic.

🚫 Smart Popup Blocker

Blocks garbage. Allows Google OAuth, CAPTCHAs, payment processors automatically. Rate limiting, size validation. Knows the difference.

📦 Download Quarantine

Downloads never touch your filesystem until you approve. 3 confirmations + 5-second delay. Heuristic scanning. Your kid can't accidentally run malware.

📡 Network Monitor

Always-visible activity indicator. Hover to see every active connection — what's connecting, bytes per second. No mystery traffic. No hidden requests.

🔑 Your Key, Not Ours

ChaCha20-Poly1305 encryption with a key YOU generate. Same crypto as WireGuard and Signal. We never see it. We never store it.

⏱️ Smart History

History doesn't save for 14.7 seconds. Click a bad link? Hit back. Gone. Stay on a page? It saves. NSFW auto-detected and excluded from history.

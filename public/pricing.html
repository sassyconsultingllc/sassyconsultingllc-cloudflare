Frequently Asked Questions

Is this really one-time? No subscription?

Correct. You buy once and own that version and every future update forever. There is no recurring charge and nothing expires. Activation lets the exe run fully offline.

How many machines can I activate?

Pro activates on 2 machines you own. Team covers up to 10. You can deactivate a machine from your Lemon Squeezy account to free a seat.

Do I need the paid tier to be useful?

No. The free tier is a complete daily driver — file ops, shell, desktop automation, persistent memory, surgical edit, and daily-driver GitHub. Pro unlocks the heavy automation surfaces (full GitHub API, Android, vision, remote Linux, system monitoring).

Which AI clients does it work with?

Any MCP client. One install auto-detects and patches Claude Desktop, Cursor, Windsurf, VS Code Copilot, Cline, Continue, Zed, and Grok Desktop — no JSON editing.

What does the Forensics add-on do, and can I add it later?

It unlocks the security-audit and registry modules (APK inspection, cert validation, hash + permission audit, Defender/firewall status, registry read/write/export, autorun forensics). It stacks on Free or Pro, and you can buy it any time.
